\section{Parameters}
This specification provides with the list of parameters for different SHRINCS instances, depending on verifier requirements:
\begin{enumerate}
    \item SHRINCS-B: size matters (instance preferable for Bitcoin).
    \item SHRINCS-L: verification complexity matters (the number of hashes to verify the signature).
\end{enumerate}

\paragraph{Note:} Parameters for stateless instance is selected in the way to support up to $2^{20}$ signatures \cite{kudinov_nick_2025_hbs_bitcoin}. \texttt{hsf} is chosen in the way that the maximum stateful signature \texttt{(at q = hsf + 1)} remains smaller than a stateless signature: \texttt{n + R\_SIZE + c + l*n + (hsf+1)*n < stateless\_size.}
