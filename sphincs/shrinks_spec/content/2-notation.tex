\section{Notation}

Throughout this specification:

\begin{itemize}
    \item[] \texttt{||} \quad denotes byte string concatenation
    \item[] \texttt{n} \quad denotes the security parameter in bits
    \item[] \texttt{H} \quad denotes a cryptographic hash function
    \item[] \texttt{Tw(P,T,m)} \quad is a a tweakable hash function with public parameter \texttt{P}, tweak \texttt{T}, and message \texttt{m}
    \item[] \texttt{PRF} \quad is a pseudorandom function
\end{itemize}
